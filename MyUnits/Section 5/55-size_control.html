Texts wrapped in P tags.

Texts wrapped in P tags. Block tag's width and height can be controlled by you in CSS.

Texts wrapped in SPAN tags. NEVER set width or height to inline tags.

Conclusion

  • Block tag's width and height can be controlled by you in CSS.
  • Inline tag's width and height are completely decided by its contents. Your width and height setting will be ignored.
  • DO NOT set width or height to inline tags.
